# Prompt
A simple television and conference prompter. 
## Technologies
* HTML5
* CSS3
* JavaScript
* jQuery
* PHP7
* MongoDB
## Quick install
* Clone this repository
* Open index.php with parameters (for the first time):
```
index.php?page=install
```
* Open index.php
* Use the Prompt
